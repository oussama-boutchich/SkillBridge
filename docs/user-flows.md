# User Flows

## Student Application Flow
1. Student signs up or logs in
2. Student completes profile
3. Student browses opportunities
4. Student opens a post
5. Student applies
6. Company receives application
7. Company accepts or declines
8. Student receives notification

## Company Posting Flow
1. Company user logs in
2. Company user completes company profile
3. Company user creates a post
4. Post becomes visible to students
5. Company user receives applications
6. Company user reviews and decides

## Admin Moderation Flow
1. Admin logs in
2. Admin opens moderation dashboard
3. Admin reviews users, posts, and logs
4. Admin bans or unbans if necessary
