# Functional Requirements

## Authentication
- User registration for students and company users
- Login and logout
- Role-based access

## Student Features
- Create and edit profile
- Upload resume link
- Add certificates
- View feed
- Browse posts
- Apply to opportunities
- Receive notifications

## Company Features
- Create and edit company profile
- Publish internship, job, and challenge posts
- View received applications
- Accept or decline applications

## Admin Features
- Log in and log out
- View users, posts, logs, analytics
- Ban and unban users

## Shared Features
- View public profiles
- View notifications
- Feed activity tracking
