# User Roles and Permissions

## Student
- Register account
- Authenticate
- Manage student profile
- Add certificates
- Browse opportunities
- Apply to posts
- View notifications and feed

## Company Owner / HR
- Register account
- Authenticate
- Manage company profile
- Create and manage posts
- View applications
- Accept or decline applications

## Admin
- Authenticate
- View all users
- View all posts
- View logs and analytics
- Ban and unban accounts
