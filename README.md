# SkillBridge Documentation Pack

This folder contains a complete Markdown-based planning and documentation starter pack for the SkillBridge project.

## Included Files
- `context.md` — main project context for Claude Code
- `memory.md` — stable project memory and key decisions
- `status.md` — current project status and progress tracking
- `project-structure.md` — recommended folder structure
- `documentation-index.md` — index of all major docs
- `prompts/` — prompts for Claude Code, database, sitemap, frontend, backend, and design
- `phases/` — project phase documents
- `docs/` — supporting planning and specification files

## Goal
Use these files as the documentation foundation of the final-year project so development stays organized, traceable, and professional.
