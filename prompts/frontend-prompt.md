# Frontend Prompt

Create the **frontend architecture and UI implementation plan** for SkillBridge using:
- HTML
- CSS
- JavaScript

## Context
SkillBridge is a professional student-career platform with three dashboards.
The design should feel modern, credible, and suitable for academic presentation.

## Your Tasks
Generate:
1. Frontend folder structure
2. Page inventory
3. Reusable UI components
4. CSS organization strategy
5. JavaScript module organization
6. Layout plan for each dashboard
7. Responsive design notes
8. Suggestions for consistent design system

## Design Direction
- Professional networking platform style
- Clean and readable dashboard layouts
- Distinct views for student, company, and admin
- Clear cards, forms, lists, tables, and notification elements

## Output Format
- Frontend architecture overview
- Folder structure
- Component list
- Per-dashboard UI plan
- Styling system recommendations
