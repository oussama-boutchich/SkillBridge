# Backend Prompt

Design the **backend API architecture** for SkillBridge using PHP and MySQL.

## Context
SkillBridge supports authentication, profiles, posts, applications, notifications, feed activities, and admin moderation.

## Your Tasks
Generate:
1. Backend folder structure
2. API module breakdown
3. Endpoint list by feature
4. Request and response conventions
5. Validation rules
6. Role-based access rules
7. Security recommendations
8. Logging and error-handling recommendations

## Required Modules
- auth
- users
- student profiles
- company profiles
- certificates
- posts
- applications
- notifications
- feed
- admin
- analytics

## Output Format
- Architecture overview
- Folder structure
- Endpoint list grouped by module
- Security and validation notes
- Implementation recommendations
