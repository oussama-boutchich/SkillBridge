# Master Prompt for Claude Code

You are helping me generate the full documentation and folder structure for my final-year IT project called **SkillBridge**.

## Project Summary
SkillBridge is a platform similar to LinkedIn, but focused on students, companies / HR recruiters, and admins.

The platform has **3 dashboards**:
1. Student dashboard
2. Company owner / HR dashboard
3. Admin dashboard

## Main Features

### Students
- Can sign up, log in, and log out
- Can create, edit, and view their profile
- Can view other profiles
- Can add certificates
- Certificate additions should create feed activities
- Can browse internships, job offers, and challenges
- Can apply to posts
- Can receive notifications when applications are accepted or declined

### Company Owners / HR
- Can sign up, log in, and log out
- Can create, edit, and view their company profile
- Can create posts for:
  - internships
  - job offers
  - challenges
- Can receive student applications
- Can accept or decline applications

### Admin
- Can log in and log out
- Can ban and unban users
- Can view status, analytics, all posts, and logs

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend API: PHP
- Database: MySQL
