# Sitemap Prompt

Create the full **sitemap and page architecture** for the SkillBridge project.

## Context
SkillBridge has public pages and 3 role-based dashboards:
- student dashboard
- company dashboard
- admin dashboard

## Your Tasks
Generate:
1. Public pages
2. Shared authenticated pages
3. Student dashboard pages
4. Company dashboard pages
5. Admin dashboard pages
6. Main navigation structure
7. Suggested page hierarchy
8. Notes about access permissions per page

## Functional Notes
- Students need profile, certificates, feed, opportunities, applications, notifications
- Companies need company profile, posts, received applications, notifications
- Admin needs analytics, user management, post review, logs, moderation

## Output Format
- Hierarchical sitemap
- Navigation groups
- Role-based access notes
- Recommendations for page priorities
