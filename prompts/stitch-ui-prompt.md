# Stitch / UI Design Prompt

Create a **frontend design brief and screen specification** for the SkillBridge platform.

## Context
SkillBridge is a LinkedIn-style platform for students and companies, with three dashboards:
- student
- company / HR
- admin

## Visual Direction
- modern
- professional
- clean
- dashboard-focused
- suitable for a final-year academic project presentation

## Key Screens Needed
- landing page
- login page
- sign up page
- student dashboard
- student profile page
- certificates page
- opportunities page
- applications page
- company dashboard
- company profile page
- create post page
- applications management page
- admin dashboard
- users moderation page
- logs / analytics page

## Your Tasks
Generate:
1. Design system direction
2. Color palette suggestions
3. Typography suggestions
4. Layout strategy
5. Component suggestions
6. Screen-by-screen descriptions
7. UX notes for forms, tables, cards, notifications, and feeds

## Output Format
- Design principles
- UI system
- Screen specifications
- Recommendations for implementation in HTML/CSS/JS
